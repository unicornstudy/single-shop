## 프로젝트 소개 📝

1인 가구 맞춤형 쇼핑몰로 기존의 쇼핑몰과 유사하게 이용이 가능하며, 구독 및 쿠폰을 통해 혜택을 받을 수 있습니다!

## 기술 스택 🏛
- Spring boot 3.0.0
- Spring Data JPA
- Spring Security
- Java17
- Mysql
- AWS EC2

## 차별점 💡
* 기능적 차별점
    * 구독시스템을 통한 무료배송 서비스 제공
    * 할인쿠폰을 통한 할인 서비스 제공
* 기술적 차별점
    * 모든 기능에 대해 테스트를 작성하여 안정적인 서비스 제공
    * 대규모 트래픽에 안전한 서비스 제공
    * 확장성을 가져 규모가 커질 경우 반찬 뿐만 아니라 다른 상품 등록까지 가능한 코드 구현
